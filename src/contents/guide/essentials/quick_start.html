Quick Start

With skylark.js sdk you can quickly create html5 applications across web and desktop.

Requirements

Skylark.js sdk is released as npm package, so first make sure that the nodejs and npm have been installed.

Installation

npm install skylarkjs -g

Create Project

enter into the workspace root directory, and run the following command to create a slax application with two views. 

sjs create slaxApp1 --routes home:/,view1:/view1,view2:/view2

Build and Deploy

1.Enter into the project root directory 

cd slaxApp1

2.run the following command to build project 

npm run build

The files builded is output to the /dist directory.
3.deploy application 

npm run deploy

The slax archive file deployed is output to the /deploy directory.
