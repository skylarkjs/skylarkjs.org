Skylark.js

An unified open source (MIT) framework

for building modern html5 applications

that across web and desktop

GET START TWITTER GITHUB

A very simple and powerful framework, all of the code is only 7k (core library, gzip) or 22k (including a powerful utility library). easy to read and debug.

The core framework focus functional structure of an application, easy to integrate with jquery, backbone, vue, react and other popular UI framework.

All applications are archived as a single .slax file which not only can be deployed to the web server, but also the local desktop.

Easy to unify other framework, such as when using skylark-jquery the jquery plugins can run directly on skylarkjs without jquery.

All constituent elements of the application such as paths, plugins are configed in single slax-config.json file.

Highly modular with very little opinion of how things should be done, removing bloat and unecessary overhead.
